Atributos

Exemplificando o conceito de atributos, são como propriedades que podem ser introduzidas aos elementos HTML para modificar o comportamento.

Atributo class

O atributo class pode ser considerado um atributo global, cujo boa parte dos elementos HTML podem ter o elemento class. Serve como definição de referência para facilitar o manuseio de scripts ou folhas de estilos.

Atributo class no Elemento a

Atributo id

O atributo id pode ser categorizado como atributo global (conceito definido anteriormente) para os elementos. Serve para facilitar a referência de scripts e/ou folhas de estilos. Porém suas declarações devem ser únicas e não pode haver mais de um elemento com o mesmo id.

Atributo id no Elemento a
