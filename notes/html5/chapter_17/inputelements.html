Introdução aos Elementos de Controle

Existem diversos elementos que pode capturar os dados introduzidos pelos próprios usuários para terem acesso as demais funcionalidades da aplicação.

Element input

O elemento input tem diversas entradas de controle de dados que pode ser definidos com o elemento input e em conjunto com o atributo type.

Logo abaixo, os elementos de entrada no geral:

Element text

Em conjunto com o valor text no atributo type no elemento input define um campo de texto.

Element checkbox and radio

Para exemplificar as caixas de seleções e botões de opções, usaremos dois elementos input cujo no primeiro elemento tem como valor no atributo type de checkbox que define caixas de seleção.

O outro elemento, tem como atributo type o valor de radio para definir um comportamento de botões de opções.

Outros elementos

Existem uma infinidades de outros elementos de entrada que serão contextualizados posteriomente.
