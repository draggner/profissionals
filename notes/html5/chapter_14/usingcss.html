Introduzir o CSS no HTML

Podemos contextualizar a inserção do CSS no HTML de diversas formas, mas a forma mais tradicional de ser utilizado o CSS, é usando o elemento link

Element link

O elemento link é uma subpropriedade de referência de um arquivo com uma definião de uso. Para declaramos um arquivo CSS, primeiro precisamos criar uma estrutura de pastas src/css/nome_arquivo.css
