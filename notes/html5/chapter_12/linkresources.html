Vinculando Recursos

Este tópico abrange diversos atributos que possam vincular um recurso específico para um comportamento especifico, como por exemplos abaixo:

Atributo charset

Este atributo associa uma codificação especifíca dependendo do valor a ser informado, como por exemplo o UTF-8.

Atributo href

Este atributo pode associar uma referência ou localização de um recurso para o documento, elementos como img, a usam esses elementos para associar seus comportamentos.

Atributo target

Esté atributo pode associar onde o documento ou elemento a ser vinculado será carregado, podendo ter diversos valores para comportamentos especificos. Elemento de uso, como o a tem essa predominância de uso deste atributo.
